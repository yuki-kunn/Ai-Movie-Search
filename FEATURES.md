# AI Movie Search - 実装機能一覧

## 🎯 主要機能

### 1. AI映画検索
- Gemini API を使用したインテリジェントな検索
- ユーザーの入力を分析し、最適な映画を推薦
- キャッシュ機能により高速なレスポンス
- サーキットブレーカーパターンでAPIエラーに対応

### 2. ユーザー認証
- **Googleアカウントでログイン**
- Firebase Authentication による安全な認証
- ユーザープロフィール管理
- ユーザー名のカスタマイズ機能

### 3. お気に入り機能
- ⭐ 星アイコンで映画をお気に入り登録
- お気に入りは自動的に塗りつぶされて視認性を向上
- お気に入り一覧ページで管理
- Firebase Firestore でデータを永続化

### 4. マイページ
- お気に入りした映画の一覧表示
- 映画カードから詳細モーダルを開く
- お気に入りから削除機能
- レスポンシブデザイン

### 5. 映画詳細表示
- 映画のポスター、タイトル、評価、公開日
- 詳細な概要説明
- 公式サイトへのリンク
- IMDb へのリンク
- 配信サービス情報（見放題/レンタル/購入）
- JustWatch へのリンク

### 6. 多言語対応
- 🇯🇵 日本語 / 🇺🇸 英語の切り替え
- モダンなタブスライドUI
- 設定を localStorage に保存
- すべてのUI要素が対応

### 7. ナビゲーション

#### PC版
- 固定ナビゲーションバー（navbar）
- 検索 / お気に入りへのリンク
- 言語切り替え
- ユーザーメニュー（設定/ログアウト）

#### スマホ版
- ハンバーガーメニュー
- スライドアウトメニュー
- コンパクトなデザイン

### 8. レスポンシブデザイン
- スマホ、タブレット、PCに完全対応
- グリッドレイアウトの自動調整
- タッチフレンドリーなUI

### 9. データ永続化
- localStorage による検索結果の保存
- ページリロード時も検索結果を保持
- 言語設定の保存
- APIを再実行せずに高速表示

### 10. ページネーション
- 検索結果を20件ずつ表示
- ページ番号ボタン
- 前へ/次へボタン
- 現在ページの表示

## 🛠 技術スタック

### フロントエンド
- **SvelteKit**: モダンなフルスタックフレームワーク
- **Svelte 5**: Runes APIを使用した状態管理
- **TailwindCSS**: ユーティリティファーストCSS
- **svelte-motion**: スムーズなアニメーション

### バックエンド
- **SvelteKit API Routes**: サーバーサイドAPI
- **Firebase**:
  - Authentication: Google認証
  - Firestore: データベース

### 外部API
- **TMDB API**: 映画データ取得
- **Gemini API**: AI分析（オプション）

## 📱 ユーザーフロー

### 初回訪問
1. トップページにアクセス
2. 言語を選択（JP/EN）
3. "Googleでログイン"をクリック
4. ユーザー名を設定（任意）

### 映画検索
1. 気分やキーワードを入力
2. "検索"ボタンをクリック
3. AI が最適な映画を推薦
4. 映画カードをクリックして詳細表示

### お気に入り管理
1. ログイン状態で映画カードの星アイコンをクリック
2. お気に入りに追加/削除
3. ナビゲーションから"お気に入り"ページへ
4. お気に入り一覧を閲覧
